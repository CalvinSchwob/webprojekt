# Webprojekt Bundy

In diesem Webprojekt sollen wir eine Infoseite zu einem Thema unserer Wahl ausarbeiten. Dazu sollen wir Standard-Webtechnologien verwenden. Die Infoseite soll gestalterisch ordentlich aussehen und keinerlei clientseitige Installationen benötigen. 

Meine Idee war es eine Infoseite zum Thema Fotografie zu erstellen. Dabei möchte ich ein wenig experimentieren sowol mit Git, also Branching etc. als auch mit interaktiven Elementen über JavaScript und UI-Design.
